\# Lab Exam - Student Loan & Payment Management System

## What this project is about
We're extending the existing Student CRUD app to add two new features: Loan Management and Payment Management. Basically, each student can now have multiple loans, and each loan can have multiple payments tied to it. The goal is for the school to be able to track loans per student and see how much of each loan has been paid off.

## Features we're building

**Loan Management**
- Students get a "Loans" button on the main table that takes you to their loan page
- You can add a loan with an Amount, Loan Type (Tuition / Books / Living Expenses), and Status (Pending / Approved / Disbursed)
- There's a view that lists all loans for that one student only

**Payment Management**
- Each loan gets a "Payments" button that opens up that loan's payment page
- You can log a payment with Amount, Date, and Method (Cash / Bank Transfer / Online Payment)
- The view shows all payments for that loan, plus how much has been paid total and what's left to pay
